# mini-toast

> A Vue.js Component

## 使用

```javascript
// 全局安装
import Vue from 'vue';
import MiniToast from 'mini-toast';
Vue.use(MiniToast)
```

```vue
this.$info.show(str);
this.$info.show(options);

this.$warn.show(str);
this.$warn.show(options);

this.$load.show(str);
this.$load.show(options);

this.$success.show(str);
this.$success.show(options);
```
